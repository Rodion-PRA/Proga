# Proga
1) Задачи(дела): основное предназаначение, вмещать в себе название необходимых задач, имеет под-окно которое будет иметь следующую структуру:
а) Удалить задачу;
b) Редактировать задачу;

2) Статус - данный раздел будет отображать текущий статус задачачи, в свою очередь данный раздел будет разделён на несколько подразделов:
а) В Процессе выполнения;
b) Выполнена;
с) Отложена;


3) Напоминание: пока не знаю как реализовать но по идеи можно выбрать (ч.м.в.), что-бы напомнить об одной или нескольких задачах, в себе содержит под-окна:
а) Изменить (ч.м.в);
b) Отменить напоминание;
с) Напомнить через (дни, часы, минуты);

4) Приоритетность: будет отображатся примерно в трёх пунктах:
a) Высокая;
b) Обычная;
с) Низкая;
